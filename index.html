Scarica il PDF

Pubblicazione su GitHub Pages

  1. Crea un repository vuoto su GitHub.
  2. Carica tutti i file di questa cartella nella root (non in una sottocartella).
  3. Vai su Settings → Pages e seleziona Branch: main / root.
  4. Apri l’URL pubblico che GitHub ti fornisce.

Incorpora in Google Sites

In Google Sites usa Incorpora → URL e incolla l’URL pubblico del tuo GitHub Pages. Puoi anche incorporare singoli blocchi di codice con Incorpora → Codice.
